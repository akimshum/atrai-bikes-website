### Introdução
Apresentamos a senseBox:bike, uma solução de hardware revolucionária projetada para capacitar os cidadãos e aprimorar a coleta de dados para um ambiente urbano mais amigável para bicicletas. Desenvolvido como parte do nosso projeto, esse hardware inovador se integra perfeitamente às bicicletas, permitindo que os indivíduos capturem dados em tempo real sobre parâmetros cruciais, contribuindo para o aprimoramento da infraestrutura para bicicletas.

### Principais recursos
1. Kit de sensores ambientais portáteis
O senseBox:bike funciona como um kit de sensor ambiental portátil, equipado com uma variedade de sensores adaptados para a coleta de dados relacionados à bicicleta. Isso inclui sensores de vibração e aceleração para avaliar a qualidade da ciclovia, um sensor de tempo de voo para identificação baseada em IA de manobras de ultrapassagem e sensores ambientais que medem fatores como material particulado e temperatura.

2. Projeto modular
Criada com base na bem-sucedida plataforma senseBox (www.sensebox.de), a senseBox:bike mantém um design modular, permitindo que os usuários adaptem e personalizem o hardware de acordo com suas necessidades específicas. A integração de slots de sensores compatíveis com QWICC, UART e GPIOs oferece flexibilidade para adicionar ou modificar sensores com base em requisitos em evolução.

3. Workshops de construção e programação fáceis de usar
Para incentivar a adoção generalizada, a senseBox:bike vem com oficinas de construção e programação fáceis de usar. Essas oficinas capacitam os cidadãos a construir e programar seus kits de sensores, promovendo um senso de propriedade e compreensão do hardware. Essa abordagem prática aumenta a motivação e o conhecimento de dados entre os participantes.

4. Medidas de privacidade de dados
De acordo com as preocupações com a privacidade, a senseBox:bike garante a proteção dos dados do usuário. Os dados coletados não são transmitidos ao vivo para a Internet, mantendo os padrões de privacidade. O dispositivo opera de maneira eficiente em termos de dados e com poucos recursos, priorizando o controle do usuário sobre seus dados.

### Implementação
A solução de hardware se baseia no senseBox desenvolvido pelo Instituto de Geoinformática da Universidade de Münster e pela re:edu, uma startup especializada em fornecer software de código aberto e recursos educacionais. A senseBox:bike expande a paleta de hardware existente, oferecendo novas oportunidades para a coleta de dados móveis com foco específico na infraestrutura de bicicletas.

### Envolvimento do cidadão
Cidades, empresas e cidadãos individuais interessados em avaliar sua infraestrutura para bicicletas podem utilizar o senseBox:bike. Os dados coletados podem ser publicados como dados abertos, contribuindo para um esforço colaborativo para melhorar a facilidade de uso da bicicleta. No entanto, o projeto enfatiza a importância de explicar o contexto do produto, abrindo a caixa preta do mundo digital para os usuários e fornecendo informações sobre o manuseio responsável dos dados.

Colaboração para soluções abrangentes
A colaboração com a 52°North, de Münster, e o Instituto Cordial, de São Paulo, aprimora a solução de hardware. Ambas as entidades são especializadas em trabalhar com dados (geográficos), introduzindo novas possibilidades de análise de dados (por exemplo, GeoAI), visualização e publicação. Juntas, elas contribuem não apenas para o desenvolvimento do hardware, mas também para a criação de um conceito abrangente de software e oficina.

### Inovações futuras
À medida que progredirmos, a senseBox:bike continuará a evoluir, incorporando o feedback dos usuários e os avanços nas tecnologias de sensores. Essa solução de hardware representa um passo tangível em direção à coleta de dados orientada para o cidadão, promovendo um senso de responsabilidade e envolvimento no desenvolvimento de ambientes urbanos mais seguros e amigáveis para as bicicletas.

### Conclusão
Em conclusão, a senseBox:bike é uma solução de hardware pioneira que capacita os cidadãos a contribuir ativamente para o aprimoramento da infraestrutura urbana de bicicletas. Com seu design modular, oficinas fáceis de usar e medidas focadas na privacidade, essa solução de hardware desempenha um papel crucial na realização das metas do nosso projeto, preparando o terreno para um futuro mais sustentável e favorável às bicicletas.
